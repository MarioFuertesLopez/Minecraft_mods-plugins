# Mods para Minecraft Java Edition

Esta carpeta contiene todos los mods organizados por loader y versión para Minecraft Java Edition.

## Loaders disponibles

- **Fabric/** - Mods para Fabric Loader
- **NeoForge/** - Mods para NeoForge Loader
- **Vanilla/** - Datapacks y recursos para Minecraft sin mods
- **Shaders/** - Packs de shaders compatibles con OptiFine e Iris
