# Biomes O' Plenty

**Descripción:** Añade más de 50 nuevos biomas al juego con bloques, plantas y árboles únicos.

**Categoría:** Contenido / Biomas

**Requisitos:** Ninguno

**Nota:** Requiere crear un mundo nuevo con el tipo de mundo "Biomes O' Plenty"

**Enlace oficial:** https://www.curseforge.com/minecraft/mc-mods/biomes-o-plenty
