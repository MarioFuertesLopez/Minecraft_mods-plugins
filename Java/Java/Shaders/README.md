# Shaders

Los shaders son packs que mejoran los gráficos de Minecraft añadiendo efectos de iluminación, sombras, reflejos y agua realista.

## Requisitos

Necesitas tener instalado uno de estos mods:
- **Iris Shaders** (para Fabric)
- **OptiFine** (para Vanilla o Forge)

## Instalación

1. Instala Iris u OptiFine según tu loader
2. Descarga el archivo .zip del shader
3. Colócalo en la carpeta `shaderpacks` dentro de `.minecraft`
4. En Minecraft, ve a Opcion
