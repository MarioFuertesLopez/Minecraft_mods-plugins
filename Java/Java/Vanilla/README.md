# Vanilla

**Aviso:** Esta sección está en construcción. Próximamente añadiremos contenido.

Minecraft Vanilla no soporta mods (.jar). Sin embargo, puedes usar:

- **Datapacks** - Modifican recetas, loot tables, estructuras y mecánicas del juego
- **Resource Packs** - Cambian texturas, sonidos y modelos

## Instalación de Datapacks

1. Descarga el datapack (.zip)
2. Abre tu mundo y ve a la carpeta `saves/[TuMundo]/datapacks`
3. Coloca el archivo .zip ahí
4. En el juego, ejecuta `/reload`

## Instalación de Resource Packs

1. Descarga el resource pack (.zip)
2. Colócalo en la carpeta `resourcepacks` dentro de `.minecraft`
3. En Minecraft, ve a Opciones > Resource Packs y actívalo

Si buscas mods que modifiquen el juego de forma más profunda, necesitarás instalar Fabric o NeoForge.
